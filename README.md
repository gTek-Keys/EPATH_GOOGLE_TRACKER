# EPATH_GOOGLE_TRACKER
This repository documents my technical development journey through the EPATH / UCLA Extension program, with a focus on real-world applications of Google Workspace, Google Cloud, Cybersecurity, and Software Engineering toward IEEE licensure and Humanitarian Computing credentials.

## 🚀 Project Vision & Completion Plan (10-Point Narrative)

1. **📌 Vision Statement**
   Build an open and modular system that tracks, proves, and showcases real-world IT professional growth aligned with **IEEE, Google Cloud, Red Hat**, and **UCLA Extension** credentialing paths.

2. **🛠️ Ground Zero**
   This project began with no institutional tools or prebuilt dashboards. The first lines of code and spreadsheets were manually crafted to address daily growth tracking and certification requirements.

3. **📚 Learning Infrastructure**
   We've created integrated tools such as a **Red Hat Study Tracker**, **7-Day Cybersecurity Audit**, and **Weekly Progress Reports** — all housed in a structured GitHub environment.

4. **📡 Cloud and Workspace Focus**
   Google Cloud and Workspace are central to our automation, reporting, and collaboration workflow. Scripts and trackers are designed for Sheets, Docs, and GCP-native services.

5. **🧠 Professional Goal**
   This repository supports the goal of becoming a **licensed IEEE Humanitarian Software Engineer** and aspiring **Semiconductor Architect**, emphasizing ethical systems and sustainable tech design.

6. **🔒 Security and Ethics**
   All tracking tools prioritize secure, offline storage (e.g., localStorage), ethical data design, and no PII transmission — aligned with humanitarian and NIST principles.

7. **🎯 Midpoint Check**
   At 50% completion, we expect to have integrated Google Cloud foundational training, completed the Red Hat tracker, and begun submitting weekly progress to Victoria Carillo.

8. **📈 Final Delivery (100%)**
   A complete professional showcase including:

   * ✅ Red Hat/React dashboard fully functional
   * ✅ Cybersecurity audit framework
   * ✅ Notion/Docs/Git reports ready for IEEE documentation
   * ✅ Coursera + LinkedIn certs mapped and tracked
   * ✅ GitHub-hosted portfolio + resume integration

9. **🌐 Collaboration Ready**
   This repo is structured for feedback and review by mentors, professors, hiring managers, and certifying organizations. PRs and README updates are encouraged.

10. **🏁 Endgame Impact**
    The final milestone is to present this repo as part of a formal IEEE or industry application — as evidence of professional-grade, humanitarian-oriented software engineering work.
